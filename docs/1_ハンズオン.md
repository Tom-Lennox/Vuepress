# 1_ハンズオン

0から初めるVuepress

```
npm install --save-dev vuepress
```

## package.json 追加
```
{
  "scripts": {
      "dev": "vuepress dev docs",
      "build": "vuepress build docs"
  },
  "devDependencies": {
      "vuepress": "^1.0.0-alpha.46"
  }
}
```

## dir構成（任意）
```
｜docs　// mdファイル置き場
　｜.vuepress
　　｜config.js　//ここにルート作る。
｜src
　｜css
　｜js
```

## ファイルの追加、削除等を行った場合
 ⇒ `yarn start`
